# prueba-tecnica-vueling

Crea una ventana emergente en la Home de Vueling para apuntarse a la newsletter.
Esta ventana deberá ser creada mediante código JS, intentando aplicar los estilos de
Vueling y, además, deberá cumplir con los siguientes requerimientos:

a. Incluir:
i. Título
ii. Texto descriptivo
iii. Input para incluir dirección de email
iv. Call to action que debe enlazar a

https://www.vueling.com/es/servicios-vueling/mas-servicios-
vueling/newsletter-vuela

b. Comportamiento:
i. Debe mostrarse después de 3 segundos
ii. Apareciendo desde el lado derecho, mediante una transición de
movimiento

c. El entregable solo debe incluir código JS que probaremos
posteriormente desde el inspector.
